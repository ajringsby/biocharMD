This repository provides a library of annealed biochar slab models and analysis code associated with _Interfacial Reactivity in Biochar-Water Systems: Insights from Reactive Molecular Dynamics_
a manuscript in preparation co-authored by Alexandra J. Ringsby and Kate Maher. 

It likewise provides analysis code for the analysis of ReaxFF bond environment, 2D surface parallel diffusion, and liquid water adsorption in biochar/water systems.
